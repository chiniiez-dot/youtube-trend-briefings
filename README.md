# youtube-trend-briefings

Operating vault for **The Art of ENTJ** — a faceless YouTube channel in the strategy-and-power lane, with a target of **$30,000/month in blended revenue** within 18–24 months.

Open the vault in [Obsidian](https://obsidian.md) (point it at the `The-Art-of-ENTJ/` folder) and start at [`The-Art-of-ENTJ/00-Home.md`](The-Art-of-ENTJ/00-Home.md).
