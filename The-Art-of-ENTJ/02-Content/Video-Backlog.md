# Video Backlog — 60 ready-to-produce ideas

Sixty titles built from the Brand Bible's headline engine, each mapped to its **pillar** (Content Strategy), **headline structure**, and **thumbnail template** (A–E). Titles follow the voice rules: statements not questions, front-loaded tension, no hype, ~40–55 characters. Pull from the top; keep this list topped up to 20+.

**Legend** — Pillars: `MR` Machiavelli Read · `NP` Named Pattern · `CS` Case Study · `OM` Operator's Manual. Templates: `A` portrait · `B` symbol · `C` versus · `D` text · `E` scene.

## How to use this list

1. Pick the next title using the [[Content-Strategy#Topic selection filter|three-part filter]] and keep the pillar ratio (~30/30/20/20) roughly intact across the month.
2. Before producing, run the [[Packaging-Playbook|three-concept workflow]] — this title is *Concept 1, the safe read*. Generate a tension play and a wildcard, then score all three.
3. Duplicate [[Video-Production-Note]] and log the title, structure, and template so [[Analytics-and-KPIs|the review]] can find patterns later.
4. Refill: whenever you pull one, add one. Sources in [[Content-Strategy#Where topics come from]].
